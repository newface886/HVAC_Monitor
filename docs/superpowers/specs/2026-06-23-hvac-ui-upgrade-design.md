# HVAC 能源监控平台 UI/UX 全面升级设计

> 为现有 WPF 工业监控平台设计一套符合科技企业工业级水准的全局视觉系统，并全面重构所有 View 页面。

**Goal:** 将现有功能完整的 HVAC 能源监控平台从简陋表格风格升级为浅色专业工业风界面：左侧图标导航、顶部状态栏、KPI 卡片、实时曲线、数字仪表盘、统计图表、统一设计系统。

**Architecture:** 不改动后端服务、ViewModel 与业务逻辑，仅在 View/XAML 层重构。引入统一 `Design/Styles.xaml` 资源字典承载颜色、字体、通用控件样式；各 View 使用 `Border`、`Grid`、`ItemsControl`、`ScottPlot.WPF` 构建布局；MahApps.Metro.IconPacks 提供图标。

**Tech Stack:** WPF (.NET 8), Prism 9, ScottPlot.WPF 5.0.35, MahApps.Metro.IconPacks

### 1.2 字体

- 标题：`Segoe UI Semibold, Microsoft YaHei Bold`
- 正文：`Segoe UI, Microsoft YaHei`
- KPI 数字：`Segoe UI Light, DIN Alternate`（大号、细字重）
- 字号：页面标题 24px，卡片标题 16px，KPI 数字 36-48px，正文 13-14px

### 1.3 通用组件样式

**Card 卡片**
- 背景：`CardBackground`
- 圆角：`8`
- 边框：`1` 像素 `Border`
- 内边距：`16`
- 轻微阴影：`DropShadowEffect ShadowDepth=2, BlurRadius=6, Opacity=0.08`

**KpiCard**
- 基于 Card
- 布局：顶部标签（`TextSecondary`，12px），中间大数字（`TextPrimary`，36px，Bold），底部单位 + 趋势箭头
- 当值越限时边框变 `Danger` 并带呼吸灯效果

**StatusBadge**
- 圆角胶囊：`14`
- 内边距：`4,8`
- 状态色：在线 `Success`、离线 `TextSecondary`、报警 `Danger`、警告 `Warning`

**IconButton / NavButton**
- 左侧导航按钮：图标 + 文字，高度 `44`，hover 背景 `PrimaryLight`，选中左侧 4px `Primary` 竖条

**SectionHeader**
- 页面标题 + 副标题 + 右侧操作区
- 标题 24px Bold，副标题 13px `TextSecondary`

**StyledDataGrid**
- 表头背景 `#F9FAFB`，文字 `TextPrimary` Bold
- 行高 `40`，悬停背景 `#F3F4F6`
- 选中行背景 `PrimaryLight`
- 边框色 `Border`

## 3. 实时监控（DashboardView）

### 3.1 布局

```
+--------------------------------------------------+
| 实时监控                              [刷新][设置]|
+--------------------------------------------------+
| [供水温度] [回水温度] [冷机功率] [风机频率] [COP] [今日能耗] |
+--------------------------------------------------+
|                     |                            |
|   实时趋势曲线       |      实时点位列表           |
|   (最近 5 分钟)      |                            |
|                     |                            |
+--------------------------------------------------+
| 最近报警： [14:23:12] 冷冻水供水温度 高限 11.5/12.0 |
+--------------------------------------------------+
```

### 3.2 KPI 卡片

六个卡片横向排列，每个卡片：
- 顶部：标签（如"冷冻水供水温度"）
- 中间：大数字（当前值，如 `7.8`）
- 底部：单位 + 趋势箭头（较上一周期上升/下降）
- 超限卡片边框变红并闪烁

KPI 数据源来自 ViewModel 的 `PointValues` 集合，按点位名称匹配。

### 3.3 实时趋势曲线

- 使用 ScottPlot.WPF
- X 轴：时间，最近 5 分钟
- Y 轴：工程值
- 同时绘制 4 条曲线：供水温度、回水温度、冷机功率、风机频率
- 不同颜色区分，带图例
- 刷新频率 1 秒，ViewModel 维护最近 300 个点

### 3.4 实时点位列表

- 表格列：点位名称、当前值、单位、质量、时间戳
- 值越限时整行变红
- 高度与趋势图一致

### 3.5 报警滚动条

- 高度 `40`，背景 `Danger` 半透明
- 显示最近 3 条未确认报警
- 带"查看全部"按钮跳转 AlarmView

## 12. 自检清单

- **Spec 覆盖度**：所有页面均给出布局、组件、样式要求；全局设计系统明确。
- **Placeholder 检查**：无 TBD/TODO/"implement later"；颜色值、尺寸、文件路径具体。
- **内部一致性**：颜色令牌在所有页面引用一致；组件名称统一。
- **Scope 检查**：本设计仅涉及 View/XAML 层，不改动后端服务与数据库模型，范围可控。
